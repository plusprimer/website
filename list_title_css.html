position属性详解

css

position属性用来做网页布局挺好用的。

position属性取值有relative，absolute，fixd, static四种

relative值表示对象不脱离文档流，其当前位置由left right top bottom四个属性来表示

absolute值表示对象脱离文档流，其参考的对象由最近的一个具有position:relative属性的对象来决定，若没有，则以body为参考。其当前位置由left right top bottom四个属性来表示

fixd值表示对象固定在一个位置，不会随着屏幕的滚动而滚动，其参考的对象以body为参考。其当前位置由left/right top/bottom的属性值来表示

static值是默认值，仍在文档流中
